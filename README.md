* Сделать так, чтобы форма добавления работы закрывалась после добавления.
* запилить контекстное меню для работы (изменить - открывается окно, в le которых уже вписаны имеющиеся значения, удалить - удаляет работу)
* убрать возможность редактирования прям из таблицы.
* сделать структуры с наименованием организаций
* сделать структуры с наименованием работников
* сделать структуру с наимнованиями этапов и отдельных работ
* при добавлении договора или этапа они должны запиливаться в app и в сериализоваться
* их всех можно выбрать из выпадающего списка или при наборе подсказываться по реливантности
