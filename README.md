# auto\_correct\_plan

Desktop-приложение для составление и корректировки документов со списком договоров

Планы по развитию проекта:

* сделать окно авторизации и выбора существующих документов на диске или создания новых для редактирования
* сделать шаблонные доки для редактирования
* разобраться и сделать обработчик файла .xlsx
* прикрутить базу данных
* запилить на основной форме список типовых задач для отдельных работ
* запилить обработку ввода в полях конкретная работа, наименование организации  наподобии поискового сервера, чтобы при вводе сразу выводились наиболее релевантные варианты
* перейти на другой Логгер, более быстрый (через <<), возможно на Boost, но это потяет за собой переход на CMake
* запилить запись логов в отдельную папку (осуществить сначала проверку существования этой папки, если нет, то создать)
* подумать на счет добавления в лог обязательной маркировки, INFO, FATAL и др. (для последующей сортировки при необходимости)
